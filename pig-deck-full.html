A proxy is not privacy

SECTION 04

The seams

two seams the mempool hid

Return path · payment

Return path. The async return path is a private mailbox, known prior art. Open: PIR-read at search scale, and scheduling the read in the auction.

Payment. Value-carrying credentials already exist. Open: composition, binding, and the funding event.

two seams it shares

Issuance cost · portability vs linkage

Issuance cost. RLN meters; it does not mint. Open: a cost that is superlinear in identities, cheap for one, anonymous at issuance.

Portability vs linkage. Conjecture: global metering and cross-egress unlinkability are mutually exclusive under any shared-nullifier construction.
